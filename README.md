# Organize Downloads Folder

This project provides a Python script to keep your Downloads folder organized. It automatically sorts files into subfolders by file type (like Images, Documents, Archives, etc.) or by date. This helps reduce clutter and makes it easier to find your files.

## Features

- Moves files into categorized subfolders based on their extension.
- Supports organization by file type or by date.
- Easy configuration for custom folder paths and rules.
- Works on Windows, macOS, and Linux.

## Getting Started

1. Clone the repository and navigate to the directory:

   ```bash
   git clone https://github.com/recklessrashdan/organize_downloads_folder.git
   cd organize_downloads_folder
   ```

2. (Optional) Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the script:

   ```bash
   python organize_downloads.py
   ```

Customize settings at the top of the script if desired. Contributions are welcome!
